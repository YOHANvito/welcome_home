About the Platform

About Welcome Home

Welcome Home is a localized rental booking system created to help renters, landlords, and administrators manage rental activities in a more organized and user-friendly way. The platform focuses on rental spaces around Sampaloc, Manila (District 4), giving users access to local listings, booking tools, comments, map view, chat, and smart property recommendations.

For Renters

Renters can browse rental listings, check prices and locations, reserve available spaces, post comments, and communicate with landlords directly through chat.

For Landlords

Landlords can post and manage their listings, update rental details, upload property images, and review booking requests from interested renters.

For Admins

Admin users can monitor platform activity, review information, manage property records, and help maintain a clean and secure rental system.

What the system offers

Welcome Home combines property search, booking management, property comments, real-time chat, landlord tools, and location-based browsing in one modern web platform. It is designed to be simple, useful, and practical for students and renters looking for nearby housing in Sampaloc, Manila.

Search and Browse

Users can explore available rentals based on title, location, amenities, and other details.

Booking System

Renters can send booking requests directly through the property page.

Map and Location

Properties can be viewed with map support for easier location checking.

Comments and Chat

Users can ask questions, leave comments, and communicate more efficiently.
